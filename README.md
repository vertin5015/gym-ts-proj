## 项目结构

```
gym-ts-proj/
 └ src/
    └ components/      # UI 组成
    └ hooks/            # 自定义 Hook
    └ pages/            # 项目页面
    └ assets/            # 项目资源
    └ App.tsx
    └ main.tsx
 └ public/
 └ index.html
 └ vite.config.js
 └ package.json
 └ ts.config.json
 └ tailwind.config.js
```

## 学习成果

- 掌握了 `Typescript` 的基本使用
- 使用 `Tailwind CSS` 加快了项目样式开发
- 加深了对响应式设计的理解
